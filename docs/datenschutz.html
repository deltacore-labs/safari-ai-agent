4. AskSafari Browser-Extension

Die AskSafari-Extension speichert API-Keys, Einstellungen und Gesprächsverläufe ausschließlich lokal im Browser-Speicher von Safari (Safari Extension Storage). Es werden keine Daten an eigene Server übertragen.

API-Anfragen werden direkt vom Browser des Nutzers an den jeweiligen KI-Anbieter gesendet (Anthropic für Claude, Google für Gemini, bzw. lokal an Ollama). Dabei gelten die Datenschutzbestimmungen des jeweiligen Anbieters. Bei Nutzung von Ollama verlassen keine Daten das lokale Gerät.
